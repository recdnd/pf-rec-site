# One-liner (Purpose)

Minimal single-page resume website for Xue Juntao (Rec), showcasing product-oriented engineering work and Spiral project portfolio.

# Audience (Who it's for)

Potential employers, collaborators, and visitors seeking to understand Rec's background, skills, and projects. Public-facing portfolio site.

# Scope (In / Out)

**In:** Personal resume presentation, project portfolio, contact information, responsive design (desktop/mobile), theme toggle, copy-to-clipboard functionality.

**Out:** Blog, dynamic content, backend services, user authentication, interactive demos, or full project documentation.

**Not included:** Spiral project source code, detailed technical specifications, or external project repositories.
